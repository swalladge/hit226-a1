Question 16 
<p class="paragraph" id="this-paragraph">What colour am I?</p>
<style>
  #this-paragraph {
    color: green;
  }
  p {
    color: red;
  }
  .paragraph {
    color: blue;
  }
</style>
The id selector gets a specificity than element or class. Therefore, the colour will be green.
