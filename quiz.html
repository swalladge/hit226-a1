About the Quiz

About

The quiz is a portable, easy to use quiz using javascript, html forms, local storage, and a canvas element.

I've licensed it under the MIT license, so go ahead and grab the main.js from this site if you want to put a quiz on your website!

Features

  • Flexible: automatically finds questions in given form element.
  • Easy to setup: A quiz object and a structured form element with html data attributes and classes means almost no coding necessary. Design the quiz in html and init the quiz object on it!
  • Uses a canvas element to display fancy feedback on submitting the quiz.
  • Gives per-question correct/wrong feedback.
  • Able to display explanations or answers if user enters wrong answer to question.
  • Can save/load questions from localStorage (supports automatic and manual saving/loading).
  • Currently works with text input, radio input, select, and checkbox input form element types.
  • Able to reset the quiz with a button.

The Javascript

The quiz is entirely powered by javascript! The code behind it takes a well structured html form element with various recognized classes, data attributes, and form elements, and transforms it into a fully working quiz.

You can view the javascript documentation for the Quiz, generated by JSDoc.
