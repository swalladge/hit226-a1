Dyslexic Fonts

The font used in this section, assuming your browser supports it, is the OpenDyslexic font (version 3). It's an open source font aimed to make it easy for people with dyslexia to read text on the computer.

Fluid Layout

This whole page is designed to be fluid and fit any screen size.

The number of columns on this page changes depending on the screen size. On a large monitor there will be multiple columns, while on a phone there will only be one column.

Fluid layouts make webpages easy to view and navigate on any screen size.

Screen Reader and Tables

Table techniques for screen reader accessibility (WebAIM 2015)
Technique Result
caption element gives a summary of what the table is about
clear headings makes it clear what each cell in the table is about
scope attribute tells the reader that a header is for a row or a column

The table above contains attributes that give a screen reader information to be able to read out the data sensibly.

Alt Attributes

Alt text set on an image via the alt attribute is used as a placeholder to explain the image if it fails to load or screen reader software is being used. (Mozilla Developer Network 2016)

CSS Media queries

Media queries are the core of responsive design. They allow certain blocks of CSS to only be processed if information about the user's device matches the query.

This site's CSS includes media queries to modify the layout if the user has a large desktop screen, average laptop size screen, or small screen (as on a phone or tablet).

For example, the following media query and contained CSS makes the navbar change to a flat layout when screen size is large. 

@media (min-width: 600px) {
  .nav-item {
    display: inline-block;
  }
}

Responsive Header

The header respond to the screen size well, changing layout or style to suit a large or small screen. If the header contains navigation links, the user must have good access to it to navigate the website.

The header on this site gives an easy to see, wide layout on large screens, but a more compact, easy to navigate version on small screens.
