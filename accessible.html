Dyslexic Font

The font used in this section, assuming your browser supports it, is the OpenDyslexic font (version 3). It's an open source font aimed to make it easy for people with dyslexia to read text on the computer.

Responsiveness

This whole webpage is designed to be responsive to almost any screen size.

The number of columns on this page changes depending on the screen size. On a large monitor there will be three columns, while on a small device (like a phone) there will only be one column.

The navbar is also designed to be usable on any screen size.

CSS Media queries

Media queries allow certain blocks of css to only be processed if information about the user's device matches the query.

This site's CSS includes media queries to modify the layout if the user has a large desktop screen, average laptop size screen, or small screen (as on a phone or tablet).

For example, the following media query and contained CSS makes the navbar change layout when screen size is small:

@media (max-width: 600px) { .nav-item {display: block;} }
